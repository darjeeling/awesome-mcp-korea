# 기여 가이드 (Contributing Guide)

## 환영합니다 (Welcome)

**awesome-mcp-korea**에 기여해 주셔서 감사합니다.
이 저장소는 한국 생태계를 위한 MCP 서버를 선별하여 제공하는 커뮤니티 기반 목록입니다.
아래 가이드를 읽고 기여해 주세요.

Thank you for contributing to **awesome-mcp-korea**.
This is a community-driven list curating MCP servers for the Korean ecosystem.
Please read the guidelines below before submitting.

---

## 추가 가능한 항목 기준 (Eligibility Rules)

아래 기준을 모두 충족하는 프로젝트만 등재할 수 있습니다.

- MCP 명세를 구현해야 합니다.
- 한국 시장 또는 한국 생태계를 대상으로 해야 합니다.
- 공개적으로 접근 가능한 저장소여야 합니다.
- 기본적인 문서(README 등)가 포함되어야 합니다.
- 장기간 비활성 상태이거나 방치된 프로젝트는 허용하지 않습니다.

Only projects meeting all of the following criteria are eligible.

- Must implement the MCP specification.
- Must target the Korean market or ecosystem.
- Must be open and publicly accessible.
- Must include basic documentation (e.g., README).
- Must not be abandoned or inactive for an extended period.

---

## Pull Request 가이드라인 (Pull Request Guidelines)

- PR 하나에 프로젝트 하나만 추가해 주세요.
- 알맞은 카테고리에 추가해 주세요.
- 알파벳 순서를 유지해 주세요.
- 아래 형식을 정확히 따라 주세요.

  ```
  **[Project Name](link)** – 한 문장의 설명.
  ```

- 설명은 반드시 한 문장으로 작성해 주세요.

- Submit one project per PR.
- Add the project under the correct category.
- Maintain alphabetical order within each category.
- Use the exact format below.

  ```
  **[Project Name](link)** – One-sentence description.
  ```

- The description must be exactly one sentence.

---

## 품질 기준 (Quality Standards)

- 중복 항목은 허용하지 않습니다.
- 마케팅 언어나 과장된 표현은 사용하지 마세요.
- 비공개 또는 접근 불가능한 프로젝트는 등재할 수 없습니다.

- No duplicate entries.
- No marketing language or exaggerated descriptions.
- No closed or inaccessible projects.

---

## 유지 관리 정책 (Maintenance Policy)

- 장기간 비활성 상태인 프로젝트는 목록에서 제거될 수 있습니다.
- 깨진 링크는 정기적으로 정리될 수 있습니다.

- Inactive projects may be removed from the list over time.
- Broken links may be cleaned up periodically.

---

## 행동 강령 (Code of Conduct)

- 모든 참여자는 서로 존중하며 소통해 주세요.
- 건설적인 피드백을 제공해 주세요.
- 모욕적이거나 공격적인 언어는 허용되지 않습니다.

- Treat all participants with respect.
- Provide constructive and helpful feedback.
- Offensive or abusive language is not tolerated.
